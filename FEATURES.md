# 一站式日程 + 饮食助手

> 给"自己当自己秘书"的高强度脑力 + 运动型个人用户的网页应用。

## 这是什么

一个跑在浏览器里的工具，把**日程安排**、**周期任务追踪**、**饮食规划**揉成一张界面。目标是替代 备忘录 + 系统日历 + 健身 app + 营养记录 app 这一堆分散工具，让一个人在没有助理的情况下也能稳定地推进工作、保持训练、吃得健康。

支持电脑和手机浏览器访问，竖屏也好看。

---

## 核心功能

### 一、智能日历 + 标签备忘

- **写一句话就生成日程**：在某一天上写下"6 月 12 号出发去旅游"，这一天就会出现一条带标签的事件
- **多种标签样式**：旅行 / 重要会议 / 截止日 / 个人事务……不同标签颜色和图标不一样，一眼看清当天性质
- **点开看详情**：日历上点任意一天，能看到这天所有事件的完整内容，像翻一页备忘录
- **不会丢**：所有事件存在你自己的数据库里，跨设备同步

> 用起来像备忘录，看起来像日历。

### 二、周期计划 → 自动落到每一天

不用每周手动填一遍重复任务。在「周期计划表」里写一次：

- 每周日 18:00 打羽毛球
- 每个工作日早 7 点跑步 30 分钟
- 每两周一次理发
- 每月 1 号交房租

→ 日历的对应日期自动出现这些任务，**可以一键打卡完成**。

- **打卡历史可追溯**：能看到这周你完成了几次健身、连续打卡多少天
- **改一次，全局更新**：把"周日打羽毛球"改成"周六"，未来所有周自动跟着改，历史打卡不动
- **支持例外**：某周临时取消、改时间，单独处理那一次，不影响整体规则

> 本质是给你自己一份「自动展开的代办清单」，让你不用反复花脑力去想"这周我该做啥"。

### 三、按当日任务联动的饮食规划

这一项是这个工具区别于普通日历的地方。它会**根据你今天日历上有什么任务**，给出对应的饮食策略：

| 当日类型 | 饮食策略示例 |
|---|---|
| **工作日**（脑力为主） | 慢碳水为主，降低餐后困倦；适量蛋白；咖啡因时机控制 |
| **运动日** / 比赛日 | 训练前补充碳水 + 电解质，训练后蛋白窗口；防止头晕、维持体能 |
| **休息日** | 重视维生素、矿物质、纤维；调整一周整体营养平衡 |

- **每日饮食一目了然**：今天三餐 + 加餐 + 补剂时间表
- **支持自定义饮食模板**：你可以定义自己的"工作日模板""比赛日模板"
- **饮食 + 任务在同一屏**：打开今天，左边任务表，右边饮食表，谁都不会落下

> 不是让你严苛节食，而是让"吃什么"这件事不再每天重新思考。

## 适合谁

- 工作内容繁杂、没有助理、需要自己长期规划的脑力工作者
- 同时有规律训练 / 健身 / 比赛习惯的人
- 想用一个界面解决"今天该做什么 + 今天该吃什么"，不愿意在多个 app 之间切换的人

## 不适合谁

- 想要团队协作 / 共享日程的人（暂未支持）
- 需要和 Google Calendar / Apple Calendar 双向同步的人（暂未支持）
- 想要原生 app 体验的人（这是个网页）

---

## 使用方式

- **桌面浏览器**：办公电脑常驻一个标签页
- **手机浏览器**：竖屏适配，路上掏出来打卡、查今日餐
- **跨设备同步**：所有数据存在你自己掌控的服务上，登录后多设备一致

---

## 隐私

这是个**自托管**应用——程序和数据都跑在你自己的设备上，不上传任何第三方服务器。你的日程、健康、饮食数据只有你自己看得到。

未来即使开放给朋友们用，也会保持这个原则。
